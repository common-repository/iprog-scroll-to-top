A simple dancing button that scrolls the page to the top automatically.

== Description ==

This plugin helps the user to move the page back to the top without scrolling their mouse or dragging the scroll bar from the browser. It is a button that appears when the user starts to scroll the page go down. Then, by clicking the button, the page should automatically goes up at the top of the page. 

This video is paid version.
https://www.youtube.com/watch?v=AYNEInyJrR8 


== Installation ==
1. Download the plugin via WordPress.org.
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard.
3. Activate the plugin through the 'Plugins' menu in WordPress.
